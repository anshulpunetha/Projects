# Zettelkasten Post-it Notes App

A lightweight browser-based Zettelkasten app where each note looks like a post-it.

## Features

- Create multiple notes
- Link notes together into threads
- Toggle view modes:
  - **Thread Notes**: selected note + all linked notes
  - **Single Note**: selected note only
- Rich text formatting for note content:
  - Font family selection
  - Font size selection
  - Bold, italic, underline
  - Bulleted lists

## Run locally

This app has no build step.

1. Open `index.html` in your browser.
2. Start creating notes and linking them.

## How linking works

Use the **Link Notes** panel to connect notes. In **Thread Notes** mode, selecting any note will show all notes connected to it (directly or indirectly).
